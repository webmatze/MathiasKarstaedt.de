My personal portfolio site
==========================

Introduction
------------

This is my personal portfolio site (http://mathiaskarstaedt.de) using nanoc and Boilerplate.

Installation
------------

1. install nanoc
    
	gem install nanoc
	
2. compile code
    
	nanoc compile
	
3. run webserver
    
	nanoc view
	
4. open in webbrowser
    
	open http://localhost:3000
